Installation of GLScene
into Embarcadero RAD Studio

The Instruction for installing GLScene library into Embarcadero RAD Studio to work with Delphi & C++Builder has include the next steps:

1. Download source codes of GLScene from the URL of repository https://svn.code.sf.net/p/glscene/code/trunk to a SVN directory on your disk, e.g. D:\Library\SVN_GLScene. Use TortoiseSVN client or RAD Studio’s embedded subversion control system in menu File | Open From Version Control… to check out the code.

You may also get the whole current Snapshot of the trunk from code page http://sourceforge.net/p/glscene/code/HEAD/tree/  or download archive zip files with previous releases of GLScene project at the page http://sourceforge.net/projects/glscene/files/

 

2. Make a copy of the trunk in a separate directory, e.g. in the working directory D:\GLScene, to prevent original sources from occasional changes. You may skip this step if you don’t need to update the code from SVN repository further.

 

3. Run setup.bat in ..\GLScene\Source\external directory as administrator to copy auxiliary dynamic libraries into C:\Windows\System32 and C:\Windows\SysWOW64 directories to support 3D sounds, shaders and physics (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton). Otherwise you will need to have ones in your application directory if exe file is calling an appropriate dll.

9. Compile GLScene’s packages for Win32/Win64.

Right click mouse button on the first line of GLScene group in Project Manager window and execute “Compile All From Here” command.

 

10. Then choose to “Install” every DesignTime package in GLScene_Win32 group to RAD Studio component palette. After installation you are ready to use GLScene library and run demos for Delphi & C++Builder in $(GLSCENEDIR)\Samples directory.
