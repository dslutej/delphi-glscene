Установка GLScene
в Embarcadero RAD Studio

Инструкция по установке библиотеки GLScene в Embarcadero RAD Studio для работы в Delphi & C++Builder. Выполните следующие шаги:

1. Загрузите исходные коды библиотеки GLScene из репозитория с URL https://svn.code.sf.net/p/glscene/code/trunk сайта http://sourceforge.net/projects/GLScene в директорию D:\SVN_GLScene на вашем диске.

Для этого можно использовать программу TortoiseSVN (команда svn checkout) или встроенную в среду RAD Studio систему поддержки версий, вызываемую из меню File | Open From Version Control…

Также можно загрузить текущую версию GLScene в виде снимка архива со страницы http://sourceforge.net/p/glscene/code/HEAD/tree/ . Для этого в правой части вверху страницы необходимо кликнуть по кнопке Snapshot.

Официальный последний релиз библиотеки можно  скачать из раздела  http://sourceforge.net/projects/glscene/files/

 

2. Скопируйте папку trunk в отдельную рабочую директорию с именем D:\GLScene, чтобы исключить в дальнейшем при работе случайное изменение исходных файлов.

 

3. Запустите командный файл setup.bat, находящийся в директории  D:\GLScene\external, с правами администратора и скопируйте в Windows\System32 и Windows\SysWOW64 вспомогательные внешние динамические библиотеки 3D звука, шейдеров и физики (BASS, FMOD, OpenAL, SDL, CG, ODE, Newton). Иначе вам придется для каждого приложения, обращающегося к этим библиотекам, помещать соответствующие DLL в директории с исполняемыми файлами. Замечание: если при запуске setup64.bat на OS Windows 10 возникнут проблемы с доступом к системным папкам, то вручную скопируйте соответствующие файлы *.dll.

6. Опции C++ Builder для файлов заголовков HPP. Откройте диалог Options в меню Tools для классического компилятора и добавьте в строке System Include path в секции Paths and Directories путь к заголовочным hpp файлам, генерируемым при компиляции исходников GLScene для Win32:

7. Опции C++ Builder для файлов библиотек BPI/LIB. Откройте диалог Options в меню Tools. Добавьте путь для доступа C++ приложений GLScene к библиотечным файлам в строке “Ordered list of Library paths” для Win32 :

 

И для Win64:

 

 

8. Откройте в меню File|Open Project…(Ctrl+F11)  групповой проект  GLScene.groupproj, находящийся в директории $(GLSCENEDIR)\Packages. В окне Project Manager должны быть перечислены следующие проекты с расширением bpl для Win32:

 

И для Win64:

 

9. Cкомпилируйте все пакеты группового проекта GLScene. Для этого в окне Project Manager щелкните правой кнопкой мыши по верхней строке проекта GLScene_RunTime и выполните команду  Compile All From Here. 

 

10. После компиляции установите из списка группового проекта DesignTime пакеты с визуальными компонентами в палитру компонентов RAD Studio с помощью команды Install. После установки компонентов GLScene можно приступить к работе с библиотекой и выполнить демонстрационные примеры для Delphi и C++Builder, находящиеся в директории $(GLSCENEDIR)\Samples.
